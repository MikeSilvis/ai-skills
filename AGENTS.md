# AI Skills Repo

- Keep skill sources under `skills/`.
- Keep command skill names lowercase and hyphenated.
- Put bulky companion material in `skills/<name>/` so Codex can load it progressively.
- Do not edit generated install targets under `~/.claude`, `~/.codex`, or `~/.cursor` directly.
- Run `dotfiles-sync --only ai-skills --force` from `~/Development/dotfiles` after changing installed skills.
